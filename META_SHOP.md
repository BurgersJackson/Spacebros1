# Meta Shop Upgrades

The Meta Shop allows you to spend "Meta Nuggets" (Nugs) collected during runs to purchase permanent upgrades. These upgrades apply to all future runs.

**Note:** The **Shop Discount** upgrade reduces the cost of all items by 10%.

## Weapon & Combat Upgrades

| Upgrade Name | Cost (Nugs) | Description | Effect |
| :--- | :--- | :--- | :--- |
| **Start Damage Boost** | 10 | Boosts starting damage output. | Multiplies damage by 1.2 (+20%). |
| **Static Blueprint** | 40 | Unlocks a permanent forward laser. | Adds a static forward-firing laser weapon. |
| **Missile Primer** | 40 | Unlocks homing missiles. | Enables homing missile capability at start. |
| **Nuke Capacitor** | 35 | Unlocks the Area Nuke ability. | Start with Nuke unlocked (500 range, 5 damage). |
| **Drone Fabricator** | 40 | Spawns a companion drone. | Starts the run with a Shooter Drone. |

## Defense & Survival

| Upgrade Name | Cost (Nugs) | Description | Effect |
| :--- | :--- | :--- | :--- |
| **Passive +HP** | 15 | Increases starting Hull Points. | +10 Max HP. |
| **Hull Plating** | 30 | Significantly increases Hull Points. | +15 Max HP. |
| **Shield Core** | 30 | Adds extra shield segments. | Adds 2 shield segments (2 HP each). |
| **Extra Life** | 60 | Grants a second chance on death. | Revive once per run with 50% HP and temporary invulnerability. |

## Utility & Economy

| Upgrade Name | Cost (Nugs) | Description | Effect |
| :--- | :--- | :--- | :--- |
| **Reroll Token** | 5 | Buy a single reroll token. | Adds +1 Reroll Token for the upgrade menu. |
| **+5 Reroll Tokens** | 25 | Buy a pack of reroll tokens. | Adds +5 Reroll Tokens for the upgrade menu. |
| **Magnet Booster** | 25 | Increases item collection range. | Sets pickup magnet radius to 300. |
| **Speed Tuning** | 25 | Increases ship movement speed. | Multiplies movement speed by 1.1 (+10%). |
| **Bank Multiplier** | 50 | Increases Nugget gain? | *Effect needs verification in code, implies better economy.* |
| **Shop Discount** | 50 | Reduces Meta Shop prices. | Reduces all future shop costs by 10%. |
